## Multi-Bar Horizontal Plot in R

### Installation

```
devtools::install_github("Kolpashnikova/package_R_MultibarHorizontal")

library(MultibarHorizontal)
```

### Description
Visualizes horizontal bars, grouped and stacked.

### Usage
MultibarHorizontal(df)
